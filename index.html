UPBEAT

ABOUT


UPBEAT is a mobile app that bridges audio and visual elements through creating Spotify playlists using uploaded images, giving each photo a unique musical element. This audio/visual package is then displayed on a map for other users to view and share their own items as well. The playlists would be influenced by the colour shift of the image, with large quantities of colours reflecting certain moods.

TARGET AUDIENCE


This app is targeted towards those with either a passion towards photography or music. Ultimaltey trying to bring more people in and create a network of users sharing their images and playlists.

PROCESS


The app would take in an image the user has taken, with this image then being pixelated and each pixel colour being recorded, the number of certain colours would link to certain moods, such as blue being linked to calmness etc. This mood value would be sent to the Spotify API in which the API would return a playlist based on the moods given as well as the users genre preferences and music tastes. This would then be stored with the image taken as the playlist image. Allowing it to be publicly shared with other users. Creating a display of audio visual work.
